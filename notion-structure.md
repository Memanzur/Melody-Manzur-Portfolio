# Notion Organization Structure for Melody

Set up these spaces/databases in Notion to organize everything.

---

## 1. Life Dashboard (Home Page)

A single page that links to everything. Quick links to each area below.

---

## 2. Projects Database

Track all your projects in one table.

| Property     | Type          | Purpose                              |
|-------------|---------------|--------------------------------------|
| Name        | Title         | Project name                         |
| Status      | Select        | Not Started / In Progress / Live / Archived |
| Category    | Select        | Product / Client Work / Learning / Personal |
| Priority    | Select        | High / Medium / Low                  |
| Due Date    | Date          | Deadline if applicable               |
| URL         | URL           | Live link or GitHub repo             |
| Notes       | Text          | Quick context                        |

**Current projects to add:**
- Boppi (Chrome Extension) - Live
- Little Red Esthetics (Client) - In Progress
- Portfolio Website - Live
- AI Academy - In Progress
- Notion Organization - In Progress

---

## 3. Digital Marketing Clients

A separate database for client work.

| Property     | Type          | Purpose                              |
|-------------|---------------|--------------------------------------|
| Client Name | Title         | Business name                        |
| Status      | Select        | Lead / Active / Completed / Paused   |
| Services    | Multi-select  | Web Design / SEO / Social / Content  |
| Monthly Fee | Number        | Revenue tracking                     |
| Start Date  | Date          | When you started                     |
| Website     | URL           | Their live site                      |
| Notes       | Text          | Important context                    |

---

## 4. AI Academy Content Planner

Track module creation for the course.

| Property     | Type          | Purpose                              |
|-------------|---------------|--------------------------------------|
| Module      | Title         | Module name                          |
| Status      | Select        | Outline / Drafting / Review / Published |
| Level       | Select        | Beginner / Intermediate / Advanced   |
| Order       | Number        | Sequence in the course               |
| Content     | Page          | The actual module content (subpage)   |

## 5. Skills & Learning Tracker

Track what you are learning and your proficiency.

| Property     | Type          | Purpose                              |
|-------------|---------------|--------------------------------------|
| Skill       | Title         | Skill name                           |
| Category    | Select        | Cybersecurity / AI / Dev / Marketing |
| Level       | Select        | Learning / Competent / Expert        |
| Resources   | URL           | Courses, docs, references            |
| Notes       | Text          | What you've learned, key takeaways   |

---

## 6. Weekly Planner

A simple weekly view to stay on track.

| Property     | Type          | Purpose                              |
|-------------|---------------|--------------------------------------|
| Task        | Title         | What needs to get done               |
| Day         | Select        | Mon / Tue / Wed / Thu / Fri / Sat / Sun |
| Category    | Select        | Work / Client / Learning / Personal  |
| Done        | Checkbox      | Mark complete                        |

---

## 7. Ideas & Notes

A quick-capture space. Dump ideas here, sort later.

- Use it like a scratchpad
- Review weekly and move items into the right database
- Tag with category so nothing gets lost

---

## How to set this up

1. Create a new page called "Life Dashboard"
2. Add linked views of each database above
3. Use the sidebar to organize: Work / Clients / Learning / Personal
4. Set up a weekly review reminder to clean up and plan ahead
